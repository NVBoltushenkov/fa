Приказ №1859/о от 30.09.2016 "Об установлении минимального количества баллов"

Минимальное количество баллов  :

 

1. Установить минимальное количество баллов единого государственного экзамена (ЕГЭ), подтверждающее успешное прохождение вступительных испытаний по общеобразовательным предметам в форме ЕГЭ, а также прохождение вступительных испытаний, проводимых Финансовым университетом для поступления на обучение по программам бакалавриата:

русский язык – 36 баллов;

математика – 27 баллов;

обществознание – 42 балла;

2. Установить минимальное количество баллов, подтверждающее успешное прохождение вступительных испытаний по образовательным дисциплинам, проводимых Финансовым университетом для поступления на обучение по программам бакалавриата по индивидуальному учебному плану лиц, имеющих соответствующее среднее профессиональное образование, - 25 баллов.

3. Установить минимальное количество баллов, подтверждающее успешное прохождение вступительных испытаний по образовательным дисциплинам, проводимых Финансовым университетом для поступления на обучение по программам бакалавриата для получения второго высшего образования, – 25 баллов.

4. Установить минимальное количество баллов, подтверждающее успешное прохождение вступительных испытаний, проводимых Финансовым университетом для поступления на обучение по программам магистратуры, - 20 баллов.
